### Clone

``` bash
$ git clone --recurse-submodules https://github.com/kira-96/kira-96.github.io.git
```

### Update Theme

``` bash
$ git submodule update --remote themes/simpleness
```

### New draft

``` bash
$ hugo new posts/my-first-post.md
```
